# Clips App

Small monorepo for a video clip management challenge.

## Stack

- Laravel 12 API (backend)
- Vue 3 + Pinia (frontend)
- Node.js microservice (independent)
- SQLite for local development

## Project Structure

- `backend/` Laravel API with Sanctum auth and clips CRUD
- `frontend/` Vue app consuming backend API
- `microservice/` Node.js service to process clip data
